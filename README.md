Code for running simulations of isotropic and anisotropic diffusion of resonance excitable media.

Made for EngBio CDT short project I, February-May 2026.
Author: Belinda Kusuma (belinda.kusuma@eng.ox.ac.uk)

To run a single simulation with given parameters, run `python3 src/app/run_single_app.py --help` and follow the instructions. 
To run multiple simulations as a way to compare metrics, run `python3 src/app/run_metric_app.py --help` and follow the instructions.
To replicate the plots shown in the poster, run `python3 data/poster/poster_analysis/plot_script.py`.
